QUICK TERM REPORT

This page lists the terms that Fisher's exact test proposes as possible explanations for the genes in the active set. Fisher's exact test is very quick: results will show up on this page instantly, as soon as you have entered some gene names on the Data page. However, be aware that Fisher's test ignores interactions between the terms, and so is generally less discriminatory than the alternative (model-based MCMC-sampling) analysis implemented by this app, which allows multiple terms to "collaborate" in explaining the data.

PARAMETERS

In strict Bayesian terminology, these counts are actually hyperparameters. Probabilistically, they represent the number of previous occurrences of various stochastic outcomes, together constituting a prior belief about what kinds of outcomes are likely. They can be thought of as weights that influence the model's tendency to impute additional events of that kind. For example, increasing the "False positive" count will tend to encourage the model not to worry so much about predicting additional genes beyond those listed in the active set.
